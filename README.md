# sales-ops-portfolio
Analysing sales pipeline performance, CRM data quality, forecast accuracy and rep performance using Excel, Power BI, Power Automate, Make.com and SQL to help businesses surface commercial insights, reduce revenue leakage and automate operational workflows.
## About

A portfolio of Sales Operations projects covering the core analytical 
and operational work a Sales Ops Analyst performs day to day. Each 
project starts from a real business problem and ends with findings 
and recommendations a sales leader could act on immediately.

Projects cover:
- Sales pipeline performance and velocity analysis
- CRM data quality auditing and governance
- Forecast accuracy modelling and weighted pipeline
- Rep performance analysis and coaching insights
- Revenue leakage identification and recovery
- Process automation using Power Automate and Make.com

- ## Tools Used Across This Portfolio

- **Microsoft Excel** — data cleaning, validation and analysis
- **Power BI** — interactive dashboards with DAX measures
- **Make.com** — automated stale deal alert workflows
- **Power Automate** — process automation and workflow management
- **SQL** — analytical queries against CRM-style data
- **Google Sheets** — live data source for automations
- **Salesforce** — CRM platform referenced across projects
